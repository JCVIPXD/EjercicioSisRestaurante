# Guia rapida de uso - SisRestaurante

## Cliente (pagina publica)

1. Entra a /SisRestaurante/index.php.
2. Revisa el menu y elige un producto.
3. Haz clic en Pedir ahora o llena el formulario.
4. Completa nombre, telefono, direccion, producto y cantidad.
5. Pulsa Enviar pedido.
6. Verifica el mensaje de confirmacion.

## Administrador

1. Entra a /SisRestaurante/admin/login.php.
2. Inicia sesion con:
   - Correo: admin@sisrestaurante.com
   - Clave: Admin2026*
3. Revisa el dashboard.
4. Administra clientes, productos, categorias, pedidos, empleados y pagos.
5. Para salir, usa Cerrar sesion.

## Si no deja ingresar

1. Verifica que la base sisrestaurante este importada.
2. Confirma que el correo y clave esten bien escritos.
3. Recarga con Ctrl + F5 e intenta de nuevo.
